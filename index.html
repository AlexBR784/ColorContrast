Como funciona el indice de contraste

El "contrast ratio" (índice de contraste) es una medida que indica cuánto contraste existe entre dos colores.

Para cumplir con las pautas de accesibilidad del W3C (WCAG 2.0), el índice de contraste mínimo recomendado para texto normal es de 4.5:1. Para texto grande y elementos no textuales, el mínimo recomendado es de 3:1 .

Por aclarar...

  • Un contraste de 1:1 significa que no hay contraste (luminancias iguales).
  • Un contraste mayor que 1:1 indica que hay contraste, y cuanto mayor sea el número, mayor será el contraste.
  • Un contraste de 21:1 significa un contraste extremadamente alto.
